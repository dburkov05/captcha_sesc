# captcha_sesc
Расширение для автоматического распознавания капчи для сайтов на основе ЭЖ "Шкала". На примере lycreg.urfu.ru

# Установка

[Аддон для firefox](https://addons.mozilla.org/ru/firefox/addon/captcha_sesc/)
